# Setup

Node v6.11.5 required

1.  `yarn install`

# Gitpod script

1. `nvm install v6.11.5`
2. `./run-github.sh`

# Run locally

1.  Start the NodeJS server: `yarn start`
2.  Start webpack: `yarn dev`
3.  Use `http://localhost:3000/` to access the app.

# Run deploying to production

1.  `yarn install`
2.  `yarn build`
3.  `yarn start`
